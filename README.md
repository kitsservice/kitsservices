## Installation

Clone the repository:

```bash
git clone https://github.com/kitsservice/kitsservices.git
```

Navigate to project:

```bash
cd kitsservices
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
